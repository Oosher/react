בנוסף :

יש תפריט ניווט sandbox הנגיש רק לאדמינים 

אדמין בלבד יכול לשנות מספר עסק בעריכה (  אם משתמש שהוא לא אדמין ינסה לשנות לא ישמר המספר )

מערכת התחברות שנועלת את המשתמש ל 24 שעות אם הקיש סיסמא לא נכונה 3 פעמים או יותר  ( הגרסה הזאת המשתמש ננעל רק ל 50 שניות לצורכי בדיקה ובשביל לא לתת לך לחכות יותר מדי זמן) ( הנעילה היא רק למשתמשים קיימים )

בדיקת ניסיונות התחברות ונעילה קוראת אך ורק בצד שרת בכדי לשמור על אבטחה גבוהה יותר
